## Overview

TaskFollowYamzy is a project built with JHipster, featuring a Spring Boot backend and an Angular frontend. It includes CI/CD setup using GitHub Actions for automated builds, tests, and SonarQube analysis.

## Getting Started

To get the project up and running locally, follow these steps:

### Prerequisites

You will need the following installed on your machine:
- Git
- Node.js (v18.x recommended)
- npm (Node Package Manager)
- Java (JDK 17 recommended)
- Maven (v3.x recommended)

### Installation

1.  **Clone the repository:**
    bash
    git clone https://github.com/YOUR_GITHUB_USERNAME_OR_ORG/TaskFollowYamzy.git
    cd TaskFollowYamzy
    

2.  **Install frontend dependencies:**
    bash
    npm install
    

3.  **Build and run the backend:**
    bash
    ./mvnw clean package -Pprod -DskipTests
    java -jar target/*.jar
    
    Alternatively, you can run it directly from Maven in development mode:
    bash
    ./mvnw
    

4.  **Run the frontend in development mode:**
    bash
    npm start
    
    The application will be available at `http://localhost:9000` (backend) and `http://localhost:4200` (frontend).

## CI/CD Pipeline

This project uses GitHub Actions for its CI/CD pipeline, defined in `.github/workflows/main.yml`. The pipeline includes:
-   Maven build (`mvnw clean package`)
-   Unit tests execution
-   Angular build (`npm run build`)
-   SonarQube analysis (requires `SONAR_TOKEN` secret to be configured in GitHub repository secrets)

The status of the latest build is displayed by the badge at the top of this README.
